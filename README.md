# EE-Book
===

## 简介
===
EE-Book是一个爬取网页内容做成EPub电子书的工具, 最初的idea来自[ZhihuHelp](https://github.com/YaoZeyuan/ZhihuHelp)，ZhihuHelp是可以帮助我们将知乎上的问题、答案、指定知乎用户的全部回答等内容爬取下来制作电子书，这个项目很有作用，也很有意思，当时我正在学习Python，于是开始跟进这个项目，后来索性写了一个新浪博客的版本，
